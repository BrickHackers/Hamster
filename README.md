Hamster
=======

Source code for the Hamster - Remote Operated Submarine.

### Structure of the project

Each board with OS has own folder with related scripts and setup


### Dependencies:

Pinebook:

* python-zmq
* python-pygame

SoPine:

* python-zmq
* python-serial

RaspberryPi:

* python-zmq
* python-serial
* darkwatterio - board 640
* darkwatterio - board Escape
